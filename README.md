# Factory Simulation

Приложение выполняет моделирование работы производственного цеха с последовательной обработкой изделий и их динамическим
распределением по станкам. Программа считывает конфигурацию цеха (матрицу времени выполнения операций) и начальное 
состояние очередей из текстового файла, путь к которому передается первым аргументом командной строки.


## Требования для сборки

- ОС: Linux
- Компилятор реализующий ISO C++20 (GCC, Clang)
- CMake (версия 3.20 или выше) или Make

## Сборка и запуск

Исполнить в корне репозитория:

```bash
mkdir build
cd build 
cmake .. 
make
```

Запуск:

```bash
./factory-simulation 123.txt
```

## Примеры ввода (файла):
```bash

3 0 1 0
2 1 0
```
